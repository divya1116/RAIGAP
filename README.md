## 📌 Overview

RAIGAP (Responsible AI Governance & Regulatory Impact Assessment Platform) is an AI-powered governance platform developed to assess Artificial Intelligence systems against Responsible AI principles, regulatory compliance requirements, and ethical standards.

The platform enables organizations to evaluate AI models for transparency, accountability, fairness, privacy, and governance while generating structured compliance reports and risk assessments.

This project was developed as part of academic research in Artificial Intelligence and Responsible AI Governance.

---

## 🎯 Objectives

- Promote Responsible AI adoption
- Evaluate AI systems using governance principles
- Assess regulatory and ethical risks
- Improve transparency and accountability
- Support AI policy and compliance decisions

---

# 🏗️ Project Workflow

1. User creates a new assessment.
2. Governance parameters are evaluated.
3. AI Ethics principles are assessed.
4. Privacy & transparency checks are performed.
5. Risk score is calculated.
6. Governance dashboard is generated.
7. Compliance report is created.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/divya1116/RAIGAP.git
```

Navigate to the project

```bash
cd RAIGAP
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
